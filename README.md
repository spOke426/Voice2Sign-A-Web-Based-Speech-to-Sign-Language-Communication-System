# Voice2Sign-A-Web-Based-Speech-to-Sign-Language-Communication-System
Voice2Sign is an innovative web-based communication system designed to bridge the gap between hearing and deaf individuals. Through the integration of speech recognition AI and animated sign language avatars, the system converts spoken words into sign language in near real time.
